# Step 4: 人物信息 + 分析标签 + 落马信息

你是一名专业的中国政治研究助理。根据官员的完整履历和人物简介，提取人物基本信息、分析标签、识别落马情况。

**输入**：
1. 官员姓名、目标省份、职务
2. 完整的履历文本（按行编号 L01/L02/...，包含起止时间-供职单位-职务的原文）
3. 人物简介文本
4. 落马相关文本（如有）

---

## raw_bio 字段说明

| 字段 | 说明 | 编码 |
|------|------|------|
| 姓名 | 官员全名 | 字符串 |
| 出生年份 | 4位整数，不确定写 0 | 整数 |
| 籍贯 | **完整省份正式名**（如"江苏省"、"新疆维吾尔自治区"） | 字符串 |
| 籍贯（市） | **完整市级正式名**（如"连云港市"），不确定写 `""` | 字符串 |
| 少数民族 | 少数民族=1，汉族=0，不确定=-1 | 0/1/-1 |
| 女性 | 女性=1，男性=0 | 0/1 |
| 全日制本科 | 有全日制本科学历=1，否则=0（党校/函授/在职不算）| 0/1 |

## 标签判断规则

### 升迁_省长（离任目标省份省长/主席后）

**定义**：离任目标省份省长（含自治区主席、直辖市市长）后，是否升迁至更高级别？

**判断步骤**：
1. 找到在目标省份担任省长/主席/市长（含代省长/代主席）的最后一条记录
   - **代省长/代主席与正式省长/主席视为同一任期延续，升迁判断以正式省长/主席离任为准**
2. 查看离任后的**下一个主要职务**（紧接的下一职务），据此判断升迁
3. **升任本省省委书记 = 1**（同省升迁也算升迁）
4. **升任其他省省委书记 = 1**
5. 下一个主要职务有晋升（省部级→副国级→正国级）= 1
6. 平级调动或降级 = 0
7. 正常退休（离任后无新职务，因年龄/届满退休）= 2
8. 任上去世或病逝 = 3
9. 仍在任（终止时间="至今"）= -1
10. **此人从未担任过目标省份省长/主席 → 留空 `""`**

### 升迁_省委书记（离任目标省份省委书记后）

**定义**：离任目标省份省委/自治区党委/市委书记后，是否升迁至更高级别？

**判断步骤**：
1. 找到在目标省份担任省委书记/自治区党委书记/市委书记（直辖市）的最后一条记录
2. 查看离任后的**下一个主要职务**（紧接的下一职务），据此判断升迁
3. 下一个主要职务有晋升（省部级→副国级/正国级）= 1
4. 平级调动或降级 = 0
5. 正常退休（离任后无新职务，因年龄/届满退休）= 2
6. 任上去世或病逝 = 3
7. 至今仍在任= -1
8. **此人从未担任过目标省份省委书记 → 留空 `""`**

### 本省提拔（到目标省份主官前）

**定义**：被任命为目标省份主官时，是否由本省体系内提拔？

**判断步骤**：
1. 找到开始在目标省份担任主官（省长或省委书记）的那条记录
2. 查看担任目标省份主官之前、时间上最接近的主要工作经历
3. 如果同时兼任多个职务，其中有任何一个在本省省级机构任职 → = 1
4. 上一条主要经历在目标省份的省级机构（如从副省长升省长）→ = 1
5. 上一条主要经历为中央/其他省份 → = 0
6. 无法判断 → = -1

### 本省学习（任前在省内全日制学习）

**定义**：在担任目标省份主官之前，是否有在目标省份内的脱产全日制学习经历？

**判断**：
1. 遍历所有早于目标省份主官任职的履历行
2. 查找：脱产全日制学习经历（关键词如"本科生""研究生""硕士研究生""博士研究生"等）+ 学习地点在目标省份境内（如目标省份的大学）
3. **不算**：在职研究生、党校进修、高校/党校任职（那是工作经历），只算全日制本科生/研究生
4. 有 → = 1，否则 = 0

## 是否落马 / 落马原因

- **是否落马**：若最终被查处落马 → `"是"`；未落马 → `"否"`
- **落马原因**：**完整摘录**所有落马表述，包括：
  - 纪委通报措辞（如"严重违纪违法"）
  - 涉及罪名（如"受贿罪"）
  - 处分决定（如"开除党籍""开除公职"）
  - 判决结果（如"判处有期徒刑14年"）
  - 涉案金额（如"受贿人民币X万元"）
  - 时间线
  - 用分号连接
- 未落马留空 `""`

---

## 重要提醒

1. **必须填写依据字段**，即使无法判断也要写理由
2. **无法判断时填 -1**，不要猜测
3. **从未担任过省长/省委书记时，对应升迁字段留空 `""`**
4. **只输出纯JSON**，不要任何解释文字
5. **依据字段控制在50字内**
6. **籍贯用完整正式省份名，籍贯（市）用完整正式市级名**
7. **落马原因必须完整摘录**，不要概括缩写
