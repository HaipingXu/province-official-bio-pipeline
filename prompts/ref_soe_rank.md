# 参考手册：国有企业的行政级别、央地属性判定

> 本文件由 Step1（组织标签 + 央地属性）和 Step3（行政级别推断）在遇到国企相关经历时引用。

## 一、国企分级体系

### 1.1 中管央企（副部级，54家）

由**中组部直接任命**一把手（董事长、总经理、书记），级别为副部级。

中国核工业集团、中国航天科技集团、中国航天科工集团、中国航空工业集团、中国船舶集团、中国兵器工业集团、中国兵器装备集团、中国电子科技集团、中国航空发动机集团、中国融通资产管理集团、中国石油天然气集团（中石油）、中国石油化工集团（中石化）、中国海洋石油集团（中海油）、国家石油天然气管网集团（国家管网）、国家电网、中国南方电网、中国华能集团、中国大唐集团、中国华电集团、国家电力投资集团、中国三峡集团、雅砻江流域水电开发有限公司（雅江）、国家能源投资集团（原神华+国电）、中国电信集团、中国联合网络通信集团（中国联通）、中国移动通信集团、中国卫星网络集团、中国电子信息产业集团、中国第一汽车集团（一汽）、东风汽车集团、中国一重集团、中国机械工业集团（国机集团）、哈尔滨电气集团、东方电气集团、鞍钢集团、中国宝武钢铁集团、中国矿产资源集团、中国铝业集团（中铝）、中国远洋海运集团（中远海运）、中国航空集团（国航）、中国东方航空集团、中国南方航空集团、中国中化控股（原中化+中国化工）、中粮集团、中国五矿集团、中国通用技术集团、中国建筑集团（中建）、中国储备粮管理集团（中储粮）、中国南水北调集团、国家开发投资集团（国投）、招商局集团、华润集团、中国旅游集团、中国商用飞机有限公司（中国商飞）

**级别对照**：
| 职务 | 级别 |
|------|------|
| 董事长/总经理/书记（中组部任命） | 副部级 |
| 副董事长/副总经理/纪检组长 | 正厅级 |
| 总会计师/总经济师/总工程师 | 正厅级（部分副厅） |
| 部门总经理/总监 | 正处级 |
| 部门副总经理 | 副处级 |

### 1.2 中央金融企业（27家，副部级）

由**财政部履行出资人职责**，一把手由中组部任命，级别为副部级。

**政策性银行（3家）**：
国家开发银行、中国进出口银行、中国农业发展银行

**国有商业银行（5家）**：
中国工商银行、中国农业银行、中国银行、中国建设银行、交通银行

**综合金融集团（2家）**：
中信集团、光大集团

**保险集团（4家）**：
中国人民保险集团（人保）、中国人寿保险集团（国寿）、中国太平保险集团、中国出口信用保险公司

**资产管理公司（4家）**：
中信金融资产管理公司（原华融）、中国长城资产管理公司、中国东方资产管理公司、中国信达资产管理公司

**主权基金与其他（9家）**：
中国投资有限责任公司（中投）、中国再保险集团、国债登记结算公司、中国农业再保险公司、中国政企合作投资基金、国家融资担保基金、中国农业信贷担保联盟、建银投资、中国银河金融控股

### 1.3 国资委管理的其他央企（正厅级）

国资委共监管约100家央企，除上述54家中管央企外，其余约46家由**国资委党委任命**一把手，级别为正厅级。

常见的正厅级央企：
中国中车集团、中国铁路工程集团（中铁）、中国铁道建筑集团（中铁建）、中国交通建设集团（中交）、中国中煤能源集团、中国节能环保集团、中国煤炭科工集团、新兴际华集团、中国诚通控股集团、中国黄金集团、中国铁塔、中林集团、中国安能建设集团、保利集团、中国建材集团、中国有色矿业集团等

**注意**：部分企业在不同时期可能从正厅级升格为副部级（如中管名单调整），以任职当时为准。

**级别对照**：
| 职务 | 级别 |
|------|------|
| 董事长/总经理/书记 | 正厅级 |
| 副董事长/副总经理 | 副厅级 |
| 部门负责人 | 正处级 |

### 1.4 省属国企（正厅级或副厅级）

由**省委组织部**或**省国资委**管理。

- **省管国企**（省委管理一把手）：正厅级
  - 典型：各省能源集团、交通投资集团、水利投资集团、省属银行等
- **省国资委管理国企**：正厅级或副厅级
  - 视具体企业规模和归口

### 1.5 市属国企（正处级至副厅级）

- **副省级城市市属国企**：副厅级（一把手副厅级）
- **普通地级市市属国企**：正处级（一把手正处级）
- **县属国企**：正科级或副处级

## 二、组织标签判定

### 央企 → 国资委央企 vs 中央金融企业

| 条件 | 组织标签 |
|------|---------|
| 国资委监管的实业类央企（54家中管 + 其余） | 国资委央企 |
| 银行、保险、证券、基金、资管类（27家） | 中央金融企业 |
| 中信集团、光大集团 | 中央金融企业 |
| 中投公司 | 中央金融企业 |
| 资产管理公司（华融/信达/长城/东方） | 中央金融企业 |

### 省属/市属国企

| 条件 | 组织标签 |
|------|---------|
| 省委/省国资委管理 | 省属国企 |
| 市委/市国资委管理 | 市属/县属国企 |

## 三、央地属性判定

### 中央

以下情况填 **"中央"**：
- 所有央企（不论总部或分公司/子公司在哪里）的**总部层面职务**
- 注意：央企在地方的**分公司/子公司**视情况：
  - 央企**省级分公司**（如中石油XX省分公司总经理）→ **地方**
  - 央企**总部部门**→ **中央**

### 地方

以下情况填 **"地方"**：
- 省属国企
- 市属/县属国企
- 央企的省级/市级分支机构

### 易混淆案例

| 原文描述 | 央地属性 | 理由 |
|----------|---------|------|
| "中石油总公司副总经理" | 中央 | 央企总部 |
| "中石油辽宁销售分公司总经理" | 地方 | 央企省级分支 |
| "国家电网公司董事长" | 中央 | 央企总部 |
| "国家电网山东省电力公司总经理" | 地方 | 央企省级分支 |
| "贵州茅台集团董事长" | 地方 | 省属国企 |
| "招商局集团董事长" | 中央 | 中管央企总部 |
| "招商银行行长" | 中央 | 中央金融企业子公司 |
| "中国华融资产管理公司董事长" | 中央 | 中央金融企业 |
| "航天部第五研究院" | 中央 | 军工央企前身 |

## 四、国企人员转政的级别认定

国企干部转任政府/党委职务时，级别认定要点：

### 4.1 常见转任路径

| 企业职务 | 级别 | 常见转任去向 |
|----------|------|-------------|
| 中管央企董事长/总经理 | 副部级 | 省长、省委书记、国务院部委部长 |
| 中管央企副总经理 | 正厅级 | 副省长、省委常委、部委副部长 |
| 省属国企董事长 | 正厅级 | 副省长、省政府秘书长 |
| 市属国企董事长（副省级市） | 副厅级 | 副市长、市局局长 |

### 4.2 判定规则

- **直接转任**：以新职务级别为准
- **"挂职"到政府**：保留原企业级别
- **企业兼任政府职务**（如"XX集团董事长，兼任省政协常委"）：就高不就低

## 五、历史沿革特别注意

### 5.1 企业改制/合并

| 时期 | 事件 | 影响 |
|------|------|------|
| 1998年前 | 许多企业为"部委下属工厂/研究所" | 组织标签选"科研院所（中央）"或"国资委央企"（视具体情况） |
| 1998-2003 | 国资委成立前，央企由各部委管理 | 以当时的管理归属为准 |
| 2003年后 | 国资委统一管理央企 | 按国资委央企处理 |
| 2018-2025 | 多家央企合并重组 | 以任职当时的企业名称为准 |
