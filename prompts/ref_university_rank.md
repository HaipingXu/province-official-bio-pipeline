# 参考手册：高校与党校的行政级别、央地属性判定

> 本文件由 Step1（组织标签 + 标志位 + 央地属性）和 Step3（行政级别推断）在遇到高校/党校相关经历时引用。

## 一、高校行政级别总览

### 1.1 副部级高校（31所，书记/校长副部级）

中管高校由**中组部直接任命**书记和校长（副部级），常务副校长/常务副书记为正厅级，其余副校级为副厅级。

### 1.2 中央部属高校完整名单（约118所）

**关键规则：所有中央部属高校均属"中央"，不论办学地点在哪个省市。** 例如厦门大学虽在福建，但属教育部直属，央地属性填"中央"；哈尔滨工业大学虽在黑龙江，但属工信部直属，央地属性填"中央"。

非副部级的中央部属高校为**正厅级**（书记/校长正厅级）。

### 1.3 地方属高校

- **省属重点高校**（如郑州大学、苏州大学、南昌大学等）：正厅级
- **省属普通本科院校**（较新设立）：正厅级或副厅级
- **高职高专院校**：副厅级或正处级

**级别对照（正厅级省属高校）**：
| 职务 | 级别 |
|------|------|
| 书记/校长 | 正厅级 |
| 副书记/副校长 | 副厅级 |
| 院长/系主任/处长 | 正处级 |
| 副院长/副处长 | 副处级 |

## 二、党校行政级别

### 2.1 中央党校（国家行政学院）

- 正部级单位
- 校长（院长）：由国务院总理兼任 → 正国级
- 常务副校长：正部级
- 副校长：副部级
- 教育长/秘书长：正厅级
- 各部/研究室主任：正厅级（部分副厅）

### 2.2 省级党校

- 正厅级单位
- 常务副校长：正厅级（校长由省委副书记或常委兼任）
- 副校长：副厅级
- 教务处长/部主任：正处级

### 2.3 市级党校

- 正处级单位（地级市）
- 校长：正处级
- 副校长：副处级

## 三、央地属性判定（极其重要）

### 核心规则：上述所有中央部属高校（约118所），不论办学地点在哪个省市，央地属性一律填"中央"。

示例：
- 厦门大学在福建省厦门市 → 但它是教育部直属 → **中央**
- 兰州大学在甘肃省兰州市 → 但它是教育部直属 → **中央**
- 哈尔滨工业大学在黑龙江省 → 但它是工信部直属 → **中央**
- 中国刑事警察学院在辽宁省沈阳市 → 但它是公安部直属 → **中央**
- 西北农林科技大学在陕西省杨凌 → 但它是教育部直属 → **中央**
- 暨南大学在广东省广州市 → 但它是中央统战部直属 → **中央**

### 3.1 填"中央"的情况

- 上述所有中央部属高校（约118所），**不论校址在北京还是在地方省份**
- 中央党校（国家行政学院）
- 中国社会科学院 / 中国科学院

### 3.2 填"地方"的情况

- **省属/市属高校**（如郑州大学、苏州大学、深圳大学等）
- 省级/市级党校
- 地方社科院

## 四、标志位判定要点

### 学习进修 vs 高校/党校任职

| 判断标准 | 标志位 | 是否算工作经历 |
|----------|--------|---------------|
| 职务含"本科生""研究生""硕士研究生""博士研究生""进修学员""访问学者""留学" | 学习进修 | 否 |
| 职务含"教授""副教授""讲师""助教""研究员""副研究员" | 高校/党校任职 | 是 |
| 职务含"校长""副校长""院长""副院长""书记""副书记""处长""系主任" | 高校/党校任职 | 是 |
| "其间"在职攻读学位（如"在职研究生""在职博士"） | 学习进修 | 否（但保留原职级别） |
| 中央党校省部级干部进修班 | 学习进修 | 否（但说明至少正厅级以上） |
| 中央党校中青年干部培训班 | 学习进修 | 否（说明厅级干部储备） |
